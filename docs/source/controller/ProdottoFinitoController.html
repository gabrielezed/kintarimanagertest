Controller: ProdottoFinitoController.java

Questo REST Controller espone gli endpoint per la gestione CRUD (Create, Read, Update, Delete) dei prodotti finiti (es. pizze, bevande). L'annotazione @RequestMapping imposta il percorso base /api/v1/prodotti-finiti per tutte le operazioni.

Endpoint API Definiti

Metodo HTTP Path Relativo Descrizione Risposta di Successo
GET / Recupera la lista di tutti i prodotti finiti. 200 OK con la lista di prodotti nel corpo.
GET /{id} Recupera un singolo prodotto finito tramite il suo ID. 200 OK con il prodotto nel corpo.
POST / Crea un nuovo prodotto finito. 201 Created con il prodotto creato nel corpo.
PUT /{id} Aggiorna un prodotto finito esistente. 200 OK con il prodotto aggiornato nel corpo.
DELETE /{id} Elimina un prodotto finito tramite il suo ID. 204 No Content con corpo vuoto.
