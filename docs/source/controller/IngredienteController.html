Controller: IngredienteController.java

Questo è un REST Controller, il punto d'ingresso per tutte le richieste API relative agli ingredienti. L'annotazione @RestController dice a Spring che questa classe gestirà richieste web e che le risposte saranno serializzate direttamente in formato JSON. @RequestMapping("/api/v1/ingredienti") imposta un percorso base per tutti gli endpoint definiti in questa classe.

Endpoint API Definiti

Metodo HTTP Path Relativo Descrizione Risposta di Successo
GET / Recupera la lista di tutti gli ingredienti. 200 OK con la lista di ingredienti nel corpo.
GET /{id} Recupera un singolo ingrediente tramite il suo ID. 200 OK con l'ingrediente nel corpo.
POST / Crea un nuovo ingrediente. I dati sono nel corpo della richiesta. 201 Created con l'ingrediente appena creato nel corpo.
PUT /{id} Aggiorna un ingrediente esistente. I dati sono nel corpo della richiesta. 200 OK con l'ingrediente aggiornato nel corpo.
DELETE /{id} Elimina un ingrediente tramite il suo ID. 204 No Content con corpo vuoto.
