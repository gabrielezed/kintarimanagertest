Controller: MagazzinoController.java

Questo controller espone le API per interagire con la logica di business del magazzino, con un percorso base /api/v1/magazzino.

Endpoint API Definiti

Metodo HTTP Path Relativo Descrizione Risposta di Successo
GET / Recupera lo stato completo di tutte le scorte in magazzino. 200 OK con la lista delle voci di magazzino.
GET /sotto-soglia Recupera solo gli ingredienti la cui scorta è al di sotto della soglia minima impostata. 200 OK con la lista delle voci di magazzino sotto soglia.
PUT /ingredienti/{ingredienteId} Aggiorna la quantità in scorta di un singolo ingrediente. Richiede un corpo JSON come {"quantita": 15.5}. 200 OK con la voce di magazzino aggiornata.
