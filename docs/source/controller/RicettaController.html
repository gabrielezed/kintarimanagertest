Controller: RicettaController.java

Questo controller gestisce le ricette come "sotto-risorsa" di un prodotto finito. L'approccio RESTful prevede che il path per accedere alla ricetta di un prodotto includa l'ID del prodotto stesso, come si vede nel @RequestMapping.

Endpoint API Definiti

Metodo HTTP Path Relativo Descrizione Risposta di Successo
GET / Recupera la lista di ingredienti per la ricetta del prodotto specificato. 200 OK con la lista di righe della ricetta.
POST / Aggiunge un ingrediente alla ricetta del prodotto specificato. 201 Created con la nuova riga della ricetta creata.
DELETE /{ricettaId} Rimuove una specifica riga (un ingrediente) dalla ricetta, usando l'ID della riga stessa. 204 No Content con corpo vuoto.
