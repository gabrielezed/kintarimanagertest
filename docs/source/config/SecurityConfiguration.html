Classe di Configurazione: SecurityConfiguration.java

Questa è la classe centrale per la configurazione di Spring Security. Utilizzando l'annotazione @EnableWebSecurity, abilita e personalizza le funzionalità di sicurezza per l'intera applicazione.

All'interno di questa classe viene definito il SecurityFilterChain, che agisce come un firewall per le richieste HTTP. La configurazione attuale definisce le seguenti regole:

  • Disabilita la protezione CSRF, adatta per API REST stateless.
  • Utilizza l'autenticazione HTTP Basic.
  • Configura l'autorizzazione basata sui ruoli per ogni endpoint, distinguendo tra accessi per ADMIN e USER.
  • Si integra con JpaUserDetailsService per caricare gli utenti dal database.
  • Definisce un PasswordEncoder (BCrypt) per la gestione sicura delle password.
  • Permette l'accesso alla console H2 e la sua visualizzazione in frame per scopi di sviluppo.
