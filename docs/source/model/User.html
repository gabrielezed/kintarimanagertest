Classe Entità: User.java

Questa entità modella un utente del sistema, fondamentale per la gestione della sicurezza. Le annotazioni @Entity e @Table la definiscono come un'entità JPA mappata sulla tabella users.

Campi della Classe

Nome Variabile Tipo Descrizione
id Long L'identificatore univoco (chiave primaria) dell'utente.
username String Il nome utente utilizzato per il login. Deve essere univoco.
password String La password dell'utente. Verrà salvata nel database in formato criptato (hash BCrypt).
roles String Una stringa che contiene i ruoli dell'utente, separati da virgola (es. "ROLE_ADMIN,ROLE_USER").
