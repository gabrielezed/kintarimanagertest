Classe: Ingrediente.java

Questa classe rappresenta un'entità del database che modella un ingrediente. È utilizzata per memorizzare le informazioni di base di ogni materia prima, come il nome, l'unità di misura con cui viene gestita e il suo costo. Le annotazioni @Entity e @Table la definiscono come un'entità JPA mappata sulla tabella ingredienti.

Campi della Classe

Nome Variabile Tipo Descrizione
id Long L'identificatore univoco (chiave primaria) dell'ingrediente. Viene generato automaticamente dal database.
nome String Il nome commerciale o descrittivo dell'ingrediente (es. "Mozzarella fior di latte").
unitaDiMisura String L'unità di base per la gestione e il costo (es. "kg", "litro", "pz").
costoPerUnita double Il prezzo d'acquisto dell'ingrediente per la sua unità di misura (es. 5.50 se costa 5,50 € al kg).
