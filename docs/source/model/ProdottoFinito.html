Classe: ProdottoFinito.java

Questa entità modella un prodotto finito, ovvero un articolo presente nel menu che viene venduto ai clienti. Esempi possono essere una "Pizza Margherita", una "Coca-Cola" o un "Tiramisù". Contiene le informazioni di base come il nome, la categoria di appartenenza e il prezzo di vendita.

Campi della Classe

Nome Variabile Tipo Descrizione
id Long L'identificatore univoco (chiave primaria) del prodotto. Viene generato automaticamente dal database.
nome String Il nome del prodotto come appare sul menu (es. "Pizza Diavola").
categoria String La categoria a cui appartiene il prodotto (es. "Pizze", "Bevande", "Dessert").
prezzoVendita double Il prezzo finale al quale il prodotto viene venduto al cliente.
