Classe DTO: IngredienteDTO.java

Questo è un Data Transfer Object (DTO). Il suo scopo è quello di trasportare i dati dell'entità Ingrediente tra il server e il client in modo efficiente. A differenza dell'entità JPA, un DTO è una classe semplice che contiene solo i dati da mostrare o da ricevere, senza logica di business o dettagli legati al database. Questo migliora la sicurezza e le prestazioni.

Campi della Classe

Nome Variabile Tipo Descrizione
id Long L'identificatore univoco dell'ingrediente.
nome String Il nome dell'ingrediente.
unitaDiMisura String L'unità di misura per la gestione a magazzino.
costoPerUnita double Il costo per singola unità di misura.
