Classe DTO: RicettaDTO.java

DTO progettato per rappresentare una singola riga della ricetta di un prodotto. È pensato per essere usato all'interno di una lista per mostrare la composizione di un prodotto. Per comodità, include non solo l'ID dell'ingrediente ma anche il suo nome e l'unità di misura, evitando al client di dover fare ulteriori chiamate.

Campi della Classe

Nome Variabile Tipo Descrizione
idLongL'ID univoco della riga di ricetta.
ingredienteIdLongL'ID dell'ingrediente associato.
ingredienteNomeStringIl nome dell'ingrediente (denormalizzato per comodità).
quantitadoubleLa quantità necessaria di ingrediente.
unitaDiMisuraStringL'unità di misura dell'ingrediente (denormalizzata per comodità).
