Interfaccia Repository: RicettaRepository.java

Questo repository è dedicato all'entità Ricetta. Oltre ai metodi CRUD standard, qui abbiamo aggiunto un metodo custom.

Metodo Custom

List<Ricetta> findByProdottoFinitoId(Long prodottoFinitoId);

Questo metodo, findByProdottoFinitoId, è un esempio della potenza di Spring Data JPA. Semplicemente dichiarando un metodo con questo nome, Spring capisce che vogliamo trovare tutte le entità Ricetta che hanno un campo prodottoFinito con un certo id. Ci permette di recuperare facilmente l'intera ricetta di un prodotto.
