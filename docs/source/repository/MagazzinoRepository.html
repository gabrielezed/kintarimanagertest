Interfaccia Repository: MagazzinoRepository.java

Repository per l'entità Magazzino, che gestisce le scorte degli ingredienti.

Metodo Custom

Optional<Magazzino> findByIngredienteId(Long ingredienteId);

Simile al repository delle ricette, abbiamo aggiunto findByIngredienteId. Questo metodo ci permette di trovare in modo efficiente la voce di magazzino associata a un specifico ingrediente tramite il suo ID. L'uso di Optional è una buona pratica perché la voce di magazzino per un dato ingrediente potrebbe non esistere, e ci costringe a gestire questo caso in modo sicuro.
