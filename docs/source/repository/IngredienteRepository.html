Interfaccia Repository: IngredienteRepository.java

Questa è un'interfaccia Repository. In Spring Boot, i repository sono il meccanismo che gestisce tutte le operazioni con il database per una specifica entità.

Semplicemente estendendo JpaRepository<Ingrediente, Long>, Spring Data JPA fornirà automaticamente a questa interfaccia tutti i metodi standard per la gestione dei dati (CRUD - Create, Read, Update, Delete). Non è necessario scrivere alcuna implementazione.

public interface IngredienteRepository extends JpaRepository<Ingrediente, Long> { }
  • Ingrediente: È l'entità che questo repository gestirà.
  • Long: È il tipo della chiave primaria (l'ID) dell'entità Ingrediente.

In futuro, se avremo bisogno di metodi di ricerca più specifici (es. "trova ingrediente per nome"), potremo semplicemente dichiararli qui e Spring li implementerà per noi.
